图像轮廓

Posted on 2023-06-09 Edited on 2023-06-11

查找并绘制轮廓

  • 学习cv.findContours()，cv.drawContours()

绘制图像的所有轮廓.

  • 待处理的源图像必须是灰度二值图。因此，在通常情况下，都要预先对图像进行阈值分割或者边缘检测处理，得到满意的二值图像后再将其作为参数使用。
  • 在OpenCV中，都是从黑色背景中查找白色对象。因此，对象必须是白色的，背景必须是黑色的。
  • 函数cv2.drawContours()会在图像image上直接绘制轮廓。

轮廓属性

  • 学习找到轮廓的特征.

特征矩

比较两个轮廓最简单的方法就是比较二者的特征矩。轮廓特征矩包含对象的全局特征(几何信息),被广泛应用在模式识别,图像识别等方面.

具体语法：
retval = cv2.moments( array, [, binaryImage] )

参数：

  • array：可以是点集，也可以是灰度图像或者二值图像。当array是点集时，函数会把这些点集当成轮廓中的顶点，把整个点集作为一条轮廓，而不是把它们当成独立的点来看待。
  • binaryImage：该参数为True时，array内所有的非零值都被处理为1。该参数仅在参数array为图像时有效。

轮廓面积

retval = cv2.contourArea(contour [, oriented])

  • retval：返回的面积值。
  • contour：轮廓。
  • oriented 是布尔型值。当它为 True 时，返回的值包含正/负号，用来表示轮廓是顺时针还是逆时针的。该参数的默认值是False，表示返回的retval是一个绝对值。

轮廓周长

retval=cv2.arcLength(curve,closed)

返回值retval是轮廓的长度（周长）。

参数：

  • curve是轮廓。
  • closed是布尔型值，用来表示轮廓是否是封闭的。该值为True时，表示轮廓是封闭的。

Hu矩

Hu矩是归一化中心矩的线性组合。 Hu矩具有旋转、缩放、平移不变性，所以经常会使用Hu距来识别图像的特征。

hu=cv2.HuMoments(m)

  • 返回值hu：表示返回的Hu矩值；
  • 参数m：是由函数cv2.moments（）计算得到矩特征值。

轮廓拟合

  • 学习轮廓近似多边形的方法

矩形包围框

绘制其中一个轮廓的矩形包围框.

retval = cv2.boundingRect(array)

  • retval：表示返回的矩形边界的左上角顶点的坐标值及矩形边界的宽度和高度。
  • array：灰度图像或轮廓。

最小矩形包围框

retval = cv2.minAreaRect(points)

  • retval：表示返回的矩形特征信息。该值的结构是（最小外接矩形的中心（x,y）,（宽度，高度），旋转角度）。
    参数：

  • points：轮廓。

注意，返回值retval是不符合函数cv2.drawContours()的参数结构要求。因此，必须将其转换为符合要求的结构，才能使用。函数 cv2.boxPoints()能够将上述返回值 retval 转换为符合要求的结构。

points = cv2.boxPoints(box)

  • points：返回值是轮廓点。能够用于函数cv2.drawContours()的参数。

  • box：是函数cv2.minAreaRect()返回值的类型的值。

圆形包围框

center,radius=cv2.minEnclosingCircle(points)

  • center：返回值，最小包围圆形的中心。
  • radius：返回值：最小包围圆形的半径。
  • points：轮廓。

椭圆包围框

retval=cv2.fitEllipse（points）

  • retval：返回值，是RotatedRect类型的值。这是因为该函数返回的是拟合椭圆的外接矩形，retval包含外接矩形的质心、宽、高、旋转角度等参数信息，这些信息正好与椭圆的中心点、轴长度、旋转角度等信息吻合。
  • points：轮廓。

最优拟合直线

cv2.fitLine()用来构造最优拟合直线，该函数的语法格式为：

line=cv2.fitLine(points,distType,param,reps,aeps)

  • line：返回值，返回的是最优拟合直线参数。（点斜式）

    斜率 k = line[1] / line[0]

    截距 b = line[3] - k * line[2]

    通过点斜式的方式来确定一条直线

  • points：轮廓。

  • distType：距离类型。拟合直线时，要使输入点到拟合直线的距离之和最小.

  • param：距离参数，与所选的距离类型有关。当此参数被设置为0时，该函数会自动选择最优值。

  • reps：用于表示拟合直线所需要的径向精度，通常该值被设定为0.01。

  • aeps：用于表示拟合直线所需要的角度精度，通常该值被设定为0.01。

最小外包三角形

函数cv2.minEnclosingTriangle()用来构造最小外包三角形。该函数的语法格式为:

retval,triangle=cv2.minEnclosingTriangle（points）

  • retval：最小外包三角形的面积。
  • triangle：最小外包三角形的三个顶点集。
  • points：轮廓。

逼近多边形

函数cv2.approxPolyDP()来构造指定精度的逼近多边形曲线。具体语法格式为：

approxCurve = cv2.approxPolyDP(curve,epsilon,closed)

  • approxCurve：返回值，逼近多边形的点集。
    curve是轮廓。
  • psilon: 精度，原始轮廓的边界点与逼近多边形边界之间的最大距离。
  • closed : 布尔型值。该值为 True 时，逼近多边形是封闭的；否则，逼近多边形是不封闭的。

轮廓凸包

  • 学习轮廓凸包等概念,凸包,凸缺陷.

什么是凸包?

  • 逼近多边形是在轮廓内部逼近轮廓
  • 凸包是在轮廓外部包围轮廓的凸多边形，是物体最外层的“凸”多边形。

逼近多边形是在轮廓内部按照指定精度逼近轮廓，是轮廓的高度近似。凸包是在轮廓外部包围原有轮廓，并且仅由轮廓上的点所构成的凸多边形。凸包的每一处都是凸的，即在凸包内连接任意两点的直线仍在凸包内。在凸包内，任意连续三个点的内角小于180°。

cv2.convexHull()函数获取轮廓的凸包。语法格式为：

hull=cv2.convexHull(points[,clockwise[,returnPoints]]）

  • hull：返回值，为凸包角点,可使用cv.polylines函数绘制多边形轮廓.
  • points：轮廓。
  • clockwise：布尔型值。该值为True时，凸包角点将按顺时针方向排列；该值为False时，则以逆时针方向排列凸包角点。
  • returnPoints：布尔型值。默认值是 True，函数返回凸包角点的x/y轴坐标；当为 False时，函数返回轮廓中凸包角点的索引。

凸包与轮廓之间的部分，称为凸缺陷。

使用函数cv2.convexityDefects()获取凸缺陷。其语法格式如下：

convexityDefects=cv2.convexityDefects（contour,convexhull）

  • convexityDefects：返回值，为凸缺陷点集。它是一个数组，每一行包含的值是[起点，终点，轮廓上距离凸包最远的点，最远点到凸包的近似距离]。

    注意，返回结果中[起点，终点，轮廓上距离凸包最远的点，最远点到凸包的近似距离]的前三个值是轮廓点的索引，所以需要到轮廓点中去找它们。

  • contour：轮廓。

  • convexhull：凸包。

  • 注意，用 cv2.convexityDefects()计算凸缺陷时，需要传入凸包作为参数。在查找该凸包时，所使用函数cv2.convexHull()的参数returnPoints的值必须是False。

测试轮廓是否是凸出

  • k = cv.isContourConvex(cnt)

图像金字塔

Posted on 2023-06-08 Edited on 2023-06-09
  • 学习图像金字塔
  • 学习cv.pyrUp()，cv.pyrDown().

一组具有不同分辨率的相同图像叫做图像金字塔.（因为当它们堆叠时，最高分辨率的图像位于顶部，最低分辨率的图像位于底部，看起来像金字塔）.

有两种金字塔

  1. 高斯金字塔

    高斯金字塔是通过不断地使用高斯金字塔滤波、采样所产生的.

    低分辨率每个像素由基础级别的5个像素的贡献与高斯权重形成。通过这样做，M×N图像变成M/2×N/2图像。因此面积减少到原始面积的四分之一,它称为Octave。类似的，在扩展时，每个级别的面积变为4倍。

    我们可以使用cv.pyrDown()和cv.pyrUp()函数找到高斯金字塔。

    cv.pyrDown向下采样,得到更大的图像.

    cv.pryUp向上采样,得到更小的图像.

  2. 拉普拉斯金字塔

    • Li = Gi - pyrUp( Gi + 1 )

    式中：

    • Li：表示拉普拉斯金字塔中的第i层
    • Gi：表示高斯金字塔中的第i层

    在每一组高斯金字塔的相邻两层之间，通过将较粗的图像在空间上插值（即上采样）得到与较细的图像相同大小的图像，再将两个图像对应位置上的像素的差作为拉普拉斯金字塔的一层。由于高斯金字塔先将原始图像通过不断卷积和下采样变换为不同分辨率的图像，所以拉普拉斯金字塔中的差分图像主要反映了边缘信息和细节信息。这些差分图像可以被用于图像增强、纹理合成、特征提取和匹配等领域。同时，拉普拉斯金字塔的应用也包括图像压缩和解压缩，在这方面，拉普拉斯金字塔的零元素可用于有效表示稀疏性高的图像。

向上采样和向下采样是相反的两种操作,由于上下采样会丢失像素值,所以两种操作不可逆.

Canny边缘检测

Posted on 2023-06-08 Edited on 2023-11-13
  • 学习Canny边缘检测cv.Canny

实现一个程序,可以使用轨迹栏调节边缘检测的阈值,观察不同阈值下图像的变化.

  • Canny Edge Detection是一种流行的边缘检测算法。它由John F. Canny发明.这是一个多阶段算法,将有以下步骤.

    1. 降噪.

      边缘检测容易受噪声影响,因此第一步是使用5x5高斯滤波器消除图像中的噪声.

    2. 寻找图像的强度梯度.

      使用Sobel核对图像水平和垂直方向滤波,获得梯度大小和方向.

    3. 非极大值抑制.

      对图像扫描,检测像素点是否是其梯度方向上的局部最大值.如果不是,则认为不是边缘的像素点,去除该像素点.

    4. 双阈值筛选边界.

      设定阈值下界（minVal）和 阈值上界（maxVal）.高于上界的确认是真的边界(强边界),低于下届的被认为不是边界,将其去除.介于二者,认为是弱边界.与强边界像素相连的弱边界被认为是边界,保留;否则认为不是边界,将其去除.

  • Sobel算子检测原理(Scharr比Sobel会有更好的结果,指定参数ksize=-1)

    本质是梯度运算.寻找一阶导数极值点.

  • Laplacian算子原理

    通过二阶导数来检测边缘.寻找二阶导数零点.

  • 如果要检测两个边缘，更好的选择是将输出数据类型保留为更高的形式，例如cv.CV_16S，cv.CV_64F等，取其绝对值，然后转换回cv.CV_8U。因为负值转换到cv.CV_8U或np.uint8时均为0,我们会因此忽略掉部分信息.

形态学转换

获取结构元素

  • 两种最基本的形态学算子是腐蚀(erosion)和扩张(dilation).
  • 形态变换是一些基于图像形状的简单操作。通常在二进制图像(灰度图)上执行。它需要两个输入，一个是我们的原始图像，第二个是决定操作性质的结构元素或内核。
  • erosion = cv.erode(img, kernel, iterations=1)  # 腐蚀 与运算 内核中不全是1清零
dilation = cv.dilate(img, kernel, iterations=1)  # 扩张 或运算 内核中有1保留
opening = cv.morphologyEx(img, cv.MORPH_OPEN, kernel)  # 开运算 先腐蚀后扩张 背景除噪
closing = cv.morphologyEx(img, cv.MORPH_CLOSE, kernel)  # 闭运算 先扩张后腐蚀 前景除噪
gradient = cv.morphologyEx(img, cv.MORPH_GRADIENT, kernel)  # 形态学梯度 = 扩张 - 侵蚀
tophat = cv.morphologyEx(img, cv.MORPH_TOPHAT, kernel)  # 顶帽 = 原图 - 开运算
blackhat = cv.morphologyEx(img, cv.MORPH_BLACKHAT, kernel)  # 黑帽 = 原图 - 闭运算

图像平滑

Posted on 2023-06-07
  • 学习使用各种低通滤镜模糊图像
  • 学习将定制的滤镜应用于图像（2D卷积）

  • 我们使用低通滤波器（LPF），高通滤波器（HPF）等对图像进行滤波。

    LPF有助于消除噪声，使图像模糊。HPF滤波器有助于在图像中找到边缘。

  • 通过将图像与低通滤波器内核进行卷积来实现图像模糊,这对于消除噪音很有用。OpenCV主要提供四种类型的模糊技术。

    • 平均.

      blur = cv.blur(img,(5,5))

      cv.blur(src, ksize[, dst[, anchor[, borderType]]]) -> dst

      • src：输入图像，可以是8位或浮点型数据。
      • ksize：滤波核大小。必须为正奇数。若为 (a,b)，则表示在 x 方向和 y 方向上的卷积核大小分别为 a 和 b。
      • dst：输出图像，与原图像有相同的大小、深度和通道数。如果此参数为空，则函数会自动创建一个与原图像相同的图像。
      • anchor：锚点位置，即卷积核的中心点。默认为 (-1, -1)，表示锚点位于卷积核中心。
      • borderType：边界模式。默认为 cv2.BORDER_DEFAULT。

    • 高斯.采用像素周围的邻域并找到其高斯加权平均值.

      内核的宽高都应为奇数.,还要指定标准差.如果仅指定sigmaX，则将sigmaY与sigmaX相同。如果两个都为零，则根据内核大小进行计算。

      高斯模糊对于从图像中去除高斯噪声非常有效。

      blur = cv.GaussianBlur(img,(5,5),0)

      cv.GaussianBlur(src, ksize, sigmaX[, dst[, sigmaY[, borderType]]]) -> dst

      • src：输入图像，可以是8位或浮点型数据。
      • ksize：高斯核的大小。必须为正奇数，且在 X 和 Y 方向上的值可以不同。如果 ksize 为 (0, 0)，则从 sigmaX 和 sigmaY 中自动计算核大小。
      • sigmaX：X 方向上的高斯标准差。
      • dst：输出图像，与原图像有相同的大小、深度和通道数。如果此参数为空，则函数会自动创建一个与原图像相同的图像。
      • sigmaY：Y 方向上的高斯标准差。如果 sigmaY 为零，则与 sigmaX 相同（即仅沿 X 方向平滑）。
      • borderType：边界模式。默认为 cv2.BORDER_DEFAULT。

    • 中值.提取内核区域下所有像素的中值，并将中心元素替换为该中值.

      内核大小应为正奇数整数。它对于消除图像中的椒盐噪声非常有效。

      blur = cv.GaussianBlur(img, 5)

      cv.medianBlur(src, ksize[, dst]) -> dst

      • src：输入图像，可以是8位或浮点型数据。
      • ksize：核的大小，必须为正奇数。
      • dst：输出图像，与原图像有相同的大小、深度和通道数。如果此参数为空，则函数会自动创建一个与原图像相同的图像。

    • 双边.对于中心像素周围不同位置（尤其是和中心像素距离较近的位置）的像素的差值进行加权.

      去除噪声的同时保持边缘清晰锐利非常有效,但是,与其他过滤器相比,该操作速度较慢.

      cv.bilateralFilter(src, d, sigmaColor, sigmaSpace[, dst[, borderType]]) -> dst

      • src：输入图像，可以是8位或浮点型数据。
      • d：表示空间高斯函数标准差，即像素之间相邻关系的高斯函数标准差。
      • sigmaColor：表示像素值相似性高斯函数标准差，即像素值差异关系的高斯函数标准差。
      • sigmaSpace：表示坐标空间中的高斯函数标准差，即在像素之间加权计算的像素距离关系上的高斯函数标准差。
      • dst：输出图像，与原图像有相同的大小、深度和通道数。
      • borderType：边界模式。默认值为cv2.BORDER_DEFAULT。

图像阈值

Posted on 2023-06-07
  • 学习简单阈值,自适应阈值,Otsu阈值.
  • 学习cv.threshold,cv.adaptiveThreshold
    简单阈值的方法

    使用不同方法的结果

  • 简单阈值手动设置阈值,对每个像素应用相同的阈值,对图像使用全局阈值.

    自适应阈值基于像素周围的小区域确定像素的阈值,对图像使用局部阈值.该方法为光照度变化的图像提供更好的结果.当设置周围的区域为无穷大时,可以观察到二值结果与otsu相似.

    otsu二值化自动确定阈值.该算法被认为是图像分割中阈值选取的最佳算法，计算简单，不受图像亮度和对比度的影响。

  • cv2.adaptiveThreshold(src, maxValue, adaptiveMethod, thresholdType, blockSize, C[, dst]) -> dst

    • 该函数返回阈值后的图像.

    • src ：8位输入图像。

    • maxValue：二值化最大值，即当像素值超过阈值时，其赋值为的最大值。

    • adaptiveMethod：自适应方法，有两个选项可选：

      • cv2.ADAPTIVE_THRESH_MEAN_C：使用邻域块的平均值作为阈值。
      • cv2.ADAPTIVE_THRESH_GAUSSIAN_C：使用邻域块的高斯加权平均值作为阈值。

    • thresholdType：二值化类型，只有两个选项可选：

      • cv2.THRESH_BINARY：大于阈值的像素点设置为最大值（常用于目标检测）。
      • cv2.THRESH_BINARY_INV：大于阈值的像素点设置为最小值（常用于生成掩模）。

    • blockSize：邻域块大小，用来计算局部阈值；必须是> 1且是奇数。一般取3、5、7等。

    • C：从计算出的平均值或高斯加权平均值中减去常数C（=x）。

    • dst：输出二值图像，与原图像大小、深度和通道数相同。

  • cv.threshold(src, thresh, maxval, type[, dst]) -> retval, dst

    • 该方法返回两个输出。第一个是使用的阈值，第二个输出是阈值后的图像。
    • src：8位单通道源图像。
    • thresh：二值化阈值。根据不同的thresholdType类型，它可能被忽略。
    • maxval：设定二值化时满足条件的像素点赋予的最大值。
    • type：二值化操作的类型。当type取值为cv.THRESH_BINARY或cv.THRESH_BINARY_INV时，func会忽略thresh参数，并自动进行OTSU二值化操作。
    • dst：用于存放输出图像的Mat对象。

  • 所有简单的阈值类型为：

    • cv.THRESH_BINARY
    • cv.THRESH_BINARY_INV
    • cv.THRESH_TRUNC
    • cv.THRESH_TOZERO
    • cv.THRESH_TOZERO_INV

图像的几何变换

Posted on 2023-06-07
  • 学习图像的几何变换:缩放,平移,旋转,仿射变换,透视变换.
  • 学习cv.warpAffine,cv.warpPerspective.

cv.warpAffine输入是2x3转换矩阵，用于仿射变换(平移+旋转+缩放).cv.warpPerspective输入是3x3转换矩阵,用于透视变换.

  • 要缩小图片，一般来说最好的插值方法是 cv.INTER_AREA，而要放大一张图片的话，一般来说效果最好的是 cv.INTER_CUBIC （速度慢）或者 cv.INTER_LINEAR （速度快一些但结果仍然不错）。

  • cv.warpAffine()函数是OpenCV库中的一个二维图像变换函数，用于将原始图像应用任意的仿射或透视变换。该函数接受三个参数：

    • img：表示要进行变换的输入图像数据，通常为灰度或RGB格式。
    • M：表示仿射或透视变换矩阵，可以通过cv.getAffineTransform或cv.getPerspectiveTransform等函数生成。
    • (cols, rows)：表示输出图像的大小，通常使用输入图像的尺寸进行设置。
    • cv.warpAffine函数的第三个参数是输出图像的大小，其形式应为(width，height)。记住width=列数(横向)，height=行数(纵向)。

  • cv.getRotationMatrix2D(center, angle, scale)函数是OpenCV库中的一个图像旋转函数，用于生成一个二维旋转变换矩阵。该函数的输入参数如下：

    • center：表示旋转中心的坐标，通常设置为(cols / 2, rows / 2)，即图像的中心点。
    • angle：表示旋转角度，可以为正数（顺时针方向）或负数（逆时针方向）。
    • scale：表示旋转后图像的缩放比例，默认值为1。

  • cv.getAffineTransform(pts1, pts2)是OpenCV中的一个函数，用于计算二维仿射变换矩阵。它接收两个参数：pts1和pts2，分别是源图像和目标图像上的三个特征点。

  • cv.getPerspectiveTransform(pts1, pts2)是OpenCV中的一个函数，用于计算透视变换矩阵。它接收两个参数：pts1和pts2，分别是源图像和目标图像上的四个特征点。
